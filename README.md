## Installation

You can install the package via Composer:

```bash
composer require oliverearl/nomiai-php
```

## Usage

```php
$skeleton = new Nomiai\Nomiai();
echo $skeleton->echoPhrase('Hello, Nomiai!');
```

## Testing

```bash
composer test
```
